Acceso Rick And Morty API con Angular.

En este proyecto he utilizado la API pública y gratuita de Rick & Morty: RickAndMorty API

Podemos mostrar la lista de todos los personajes, buscar por personaje, ir a una página concreta (incluye paginación) y mostrar el detalle de cada uno de los personajes. La aplicación está desplegada en Firebase y se puede probar en esta url: RickAndMorty

FRONTEND: Angular y Bootstrap.
